# agefy-pack
Um sistema criado para desenvolvedores Brasileiros verificarem a idade de seus usuários sem ter que gastar nada.

## PACOTES
- tesseract.js
- sharp
- uuid
- cpf-cnpj-validator


## Observações
O AgeFy é um sistema desenvolvido usando Tesseract.Js, desde então, já é avisado que o AgeFy pode cometer erros de leitura de documentos, podendo recusar o documento dependendo da qualidade.
